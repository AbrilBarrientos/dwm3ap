  • Software:

    Conjunto de instrucciones y reglas para ejecutar tareas.
    Ejemplos: Windows, Photoshop, un navegador web.

  • Hardware:

    Son las partes físicas de la computadora.
    Ejemplos: procesador, memoria, teclado, monitor.

  • Programas de aplicación:

    Programas que se usan para hacer tareas específicas.
    Escribir en Word, navegar en Chrome, editar imágenes en Photoshop.

  • Recursos del sistema:

    Elementos que usa la computadora para funcionar.
    Memoria RAM, procesador, almacenamiento, dispositivos conectados.

  • Servicios del sistema operativo:

    Funciones que el sistema operativo ofrece a los programas.
    Abrir y guardar archivos, ejecutar procesos, gestionar, la memoria, permitir la conexión a internet.

Sistema Operativo

  • Un sistema operativo es un PROGRAMA ESENCIAL que permite que una computadora funcione.
  • Su función principal incluye la administración de recursos del sistema, coordinar y controlar el hardware, y proporcionar servicios a los programas de aplicación.
  • "Puente entre el usuario, los programas y el hardware."
  • Algunos ejemplos de sistemas operativos son Windows, macOS, Linux, Android e iOS.

Electron JS (framework)

  • Empaquetador que permite crear aplicaciones de escritorio con html, css y js.
  • Usa Node.js para interactuar con el sistema operativo (acceder a archivos o ejecutar procesos).
  • Ventaja: Fácil de desarrollar.
  • Desventaja: Puede consumir más recursos que una app nativa.

Historia de Linux

El nombre completo es GNU Linux

1983 - GNU

  • Richard Stallman molesto porque el software era privado y de pago, decidió crear un sistema operativo libre y gratuito llamado GNU (GNU's Not Unix).
    Quería lograr un sistema similar a Unix y a los estándares de POSIX1, pero que sea accesible para todos.

1985 - Free Software Foundation (FSF)

  • Stallman crea la Fundacion del Software Libre para apoyar el desarrollo de Software libre, público y gratuito.
  • La fundó para proteger y promover programas que cualquiera pueda usar, modificar y compartir sin restricciones.
  • La FSF creó la Licencia Pública General (GPL) de GNU. Esta licencia hace que los usuarios tengan la libertad de ejecutar, copiar, distribuir, estudiar, modificar y mejorar el software.
  • Linux usa la GPL, lo que permite que cualquiera lo mejore sin volverlo privado.

1990 - Software disponible

  • Ya existían muchos programas libres para crear un sistema operativo, pero faltaba lo más importante: el kernel.
Kernel
  • Núcleo del sistema operativo, el componente principal que se encarga de la interfaz entre el hardware y el software.
  • Sin el kernel los programas no pueden comunicarse con la computadora.
  • Se ejecuta en modo privilegiado, siempre tiene control total sobre el sistema.

Máquinas virtuales

  • Software que simula una computadora.
  • Tiene su propio sistema operativo, disco, memoria, red, etc.
  • Se ejecuta dentro de un hipervisor(ej: VirtualBox) sobre un sistema operativo anfitrión.

Hipervisor

  • Software que gestiona las máquinas virtuales.
  • Permite que varias máquinas virtuales funcionen en una sola maquina física.
  • Existen dos tipos:
    • Corre directo sobre el hardware
    • Corre sobre un sistema oeprativo
