Storage

Objeto STORAGE = API de almacenamiento web local sin base de datos.

Storage método LocalStorage

  • Los datos permanecen indefinidamente o hasta que se elminen.
  • Los datos que se guardan son pares de clave / valor en el navegador.
  • Comparte datos entre todas las pestañas y ventanas del navegador del mismo orgien.

    * cualquiera puede acceder al local storage por lo que NO SE COLOCA INFORMACION SENSIBLE

  • Los datos no expiran, persisten a los reinicios del navegador y hasta del sistema operativo.

Storage método SessionStorage

  • Los datos permanecen hasta que se cierra la pestaña del navegador

Métodos y propiedades de LocalStorage y SessionStorage

  • setItem(clave, valor)
    • Almacena un par clave/valor
  • getItem(clave)
    • Obtiene el valor por medio de la clave
  • removeItem(clave)
    • Elimina la clave y su vlaor
